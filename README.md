# 🛡️ Phishing Awareness Training

A web-based cybersecurity awareness training module developed as part of the **CodeAlpha Cyber Security Internship**.

This project educates users about phishing attacks, fake emails, fake websites, social engineering techniques, and online safety best practices through interactive lessons and a final assessment.

---

## 📚 Features

- 🏠 Home page introducing phishing awareness
- 🎣 What is Phishing?
- 📧 Identifying Fake Emails
- 🌐 Recognizing Fake Websites
- 🧠 Understanding Social Engineering
- 🛡️ Cyber Safety Tips
- 📝 Interactive Final Quiz
- 👤 Learner name input
- 📊 Automatic quiz scoring
- ✅ Pass/Fail assessment
- 🔄 Quiz retake option
- 🏆 Personalized Certificate of Completion

---

## 🚀 How to Run

1. Download or clone the repository.

2. Open the project folder in Visual Studio Code.

3. Open `index.html` using Live Server.

4. Navigate through the lessons and complete the final quiz.

5. Pass the quiz to receive a personalized certificate.

---

## 🎯 Learning Objectives

After completing this training, users should be able to:

- Understand phishing attacks.
- Identify fake emails.
- Recognize fraudulent websites.
- Understand social engineering techniques.
- Apply cybersecurity best practices.
- Complete a phishing awareness assessment.
